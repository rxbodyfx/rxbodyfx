# RxBODYFx Website (Rebuild)

Next.js static-export site for RxBODYFx: medical weight loss, hormone
optimization, nutrition coaching, and telehealth (nationwide, via Asher
Health partnership).

## Local dev
npm install
npm run dev

## Build (static export to /out)
npm run build

## Deploy
- Vercel: import this repo, no config needed (next.config.js already set to output: 'export').
- GitHub Pages: serve the generated /out directory after `npm run build`.
